Projects

Pneumonia Detection with CNNs

This computer vision project utilized chest X-ray images to detect pneumonia. Leveraging convolutional neural networks (CNNs) and transfer learning, I developed a model that achieved 91% accuracy in detecting pneumonia from X-ray images. Techniques such as data augmentation were also employed to improve performance. The project is part of my ongoing work in applying machine learning to healthcare.

Speechsense App

The Speechsense App combines automatic speech recognition (ASR) and text-to-speech (TTS) technology with emotion detection. It recognizes emotions such as joy, sadness, anger, and surprise, adding depth to sentiment analysis beyond positive, negative, and neutral sentiments. The app's user interface was designed using Gradio, making it intuitive and accessible for users.

Comparative Analysis of Models for Time Series Forecasting

This project focused on comparing ARIMA, XGBoost, and LSTM models for time series forecasting, applied to Bitcoin price data. The analysis highlighted that LSTM is the best model for volatile and non-linear data. The project explored various aspects of feature engineering, model tuning, and performance evaluation, with detailed insights into each model’s strengths and weaknesses.

Chicago Crime Data Analysis

This project analyzes crime data from Chicago for the year 2022, utilizing a dataset sourced from Kaggle, which contains 219,032 records and 22 columns. The focus of the analysis is on identifying trends and crime hot spots, with a particular emphasis on the most common crimes, arrest rates, and crime distribution across various wards, districts, and community areas. Visualizations provide insights into crime patterns throughout the year, including fluctuations based on days of the week and seasonal trends.

Interactive Dashboard: Chicago Crime Data Analysis

Explore the interactive Tableau dashboard showcasing the analysis of crime data from Chicago in 2022. This dashboard allows users to delve into key aspects of crime reporting in the city, offering a visual representation of the data for a comprehensive understanding of trends and patterns.

UEFA Champions League Statistics Dashboard

This project features an interactive dashboard that presents key statistics about the UEFA Champions League tournament, showcasing its rich history and significant milestones. Designed for football enthusiasts and analysts alike, this dashboard provides a comprehensive overview of the tournament’s most important data.
